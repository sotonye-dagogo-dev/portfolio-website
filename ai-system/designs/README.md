## Visual System

- **Display:** `--font-display: 'Canela', 'Ampersand', 'Iowan Old Style', Georgia, serif` — heroic headings, massive scale
- **Body:** `--font-body: 'Inter', -apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif` — UI, navigation, body text
- **Mono:** `--font-mono: 'JetBrains Mono', 'SF Mono', Menlo, monospace` — labels, tags, code

### Key effects

- Horizontal scroll galleries with `scroll-snap` (projects, certificates)
- Marquee with CSS `linear infinite` animation (tech stack, logos)
- Custom cursor — 16px dot, `mix-blend-mode: difference`, scales on hover, hidden on touch
- Theme toggle — `[data-theme]` + `localStorage` persistence
- Back-to-top — fades in after 400px scroll
- `IntersectionObserver` scroll reveals: `.reveal` (fade up), `.reveal-left`, `.reveal-right`, `.stagger-children`

### Mobile Responsiveness

- Navbar converts to slide-in drawer (right side) with hamburger + overlay backdrop
- Horizontal galleries are natively scrollable via touch; nav buttons provided for desktop
- All responsive breakpoints at 900px, 768px, 480px
- Custom cursor hidden on touch devices

### Implementation Notes

- Global CSS in `src/styles.scss` is single source of truth for all theme tokens, utility classes, animations, and responsive rules
- Component SCSS files only contain page-specific styles
- Tech stack pills in home page marquee use SVG icons from `public/techStackIcons/`
- Profile picture (`public/profile-picture.jpg`) used as hero background on home page
- Automation flow nodes no longer mark "Open Design" as active
- Archive section uses CSS `max-height` transition for accordion behavior
