# Designs

Visual design contracts for the portfolio-website revamp.
Each file is a self-contained HTML document — no framework dependencies.
Generated by Open Design; consumed by Open Code for implementation.

## Design System

| File | Description |
|------|-------------|
| `od-system.html` | Canonical design system: theme tokens (light/dark), typography, spacing, components, interaction directives reference |

## Visual System

- **Fonts:** Inter (sans-serif / body), Playfair Display (serif / headings), JetBrains Mono (mono / tech labels)
- **Accent:** `#d4a040` (gold)
- **Theme:** Full dark/light support via `[data-theme]` CSS custom properties
- **Key effects:** Marquee (tech stack), horizontal slider galleries (projects, certificates), back-to-top button, 3D card hover, parallax sections, scrolling text

## Interaction Affordances (Angular directives)

| Directive | Applied To | Visual Cue in OD Files |
|-----------|-----------|------------------------|
| `typing-effect` | Hero tagline (OD-1), philosophy strip (OD-5), subtitle (OD-6) | Blinking cursor via `border-right` animation |
| `animated-border` | Primary CTA (OD-1), flagship cards (OD-4), action buttons (OD-6) | Rotating conic-gradient border via `::before` |
| `pulsating-effect` | Availability badge (OD-1), current role dots (OD-3), automation badge (OD-4), active flow node (OD-5) | Scale + opacity glow loop on dot |
| `image-viewer` | Project thumbnails (OD-4), certificate thumbnails (OD-6) | Click-to-expand overlay, cursor:zoom-in |

## Config Mapping

Every copy string in each OD file is wrapped in `[CONFIG:path]` markers
that map 1:1 to fields in `src/app/content/site.config.ts`.
