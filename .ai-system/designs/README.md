# Designs

Visual design contracts for the portfolio-website redesign.
Each file is a self-contained HTML document — no framework dependencies.
Generated by Open Design; consumed by Open Code for implementation.

## Design Tokens

Colors, typography, and spacing follow the values in `.ai-system/design-system.md`
and `tailwind.config.js`. Each OD file declares its own CSS custom properties
matching those tokens.

## Interaction Affordances (Angular directives)

| Directive | Applied To | Visual Cue in OD Files |
|-----------|-----------|------------------------|
| `typing-effect` | Hero tagline | Blinking cursor + char-by-char reveal annotated in CSS comments |
| `animated-border` | Primary CTA, flagship project cards | Rotating conic-gradient border via `::before` pseudo-element |
| `pulsating-effect` | "Available for work" badge | Scale + opacity loop on dot + ring glow |
| `image-viewer` | Project thumbnails | Click-to-expand overlay (annotated, not implemented in HTML) |

## Config Mapping

Every copy string in each OD file is wrapped in `[CONFIG:path]` markers
that map 1:1 to fields in `src/app/content/site.config.ts`.
