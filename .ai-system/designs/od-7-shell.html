⚙ Shared shell contract: The nav, footer, and theme toggle shown above (and repeated in every OD file) are the same across all pages. Only the .active class changes to match the current route.

Nav Integration Notes

Active State

The .active class on the current route's <a> sets color: var(--color-accent) and font-weight: 700, plus the underline ::after is always at full width. In Angular, use routerLinkActive="active".

Responsive

On screens ≤ 640px, the nav items wrap using flex-wrap: wrap and gap reduces to --space-md. No hamburger menu — the six items fit comfortably on a single wrapped line on mobile.
